## Instrucciones para instalar el proyecto

Primero debemos descargar Node.js de la [página oficial](https://nodejs.org/es/) posteriormente se instala sin ninguna especificación, abrir la consola de comandos `(cmd, powershell u otras)` e instalar `npm` con el siguiente comando `npm install -g npm@latest`, a continuación borramos la cache de npm con `npm cache clean --force`, seguidamente una vez se nos hayan brindado los permisos necesarios para acceder al repositorio de Git Lab procedemos con `git clone https://gitlab.com/kiero-developers/front-kiero.git`, ingresamos a la carpeta descargada desde la consola hasta encontrar el archivo `package.json` y las carpetas del proyecto, una vez allí ejecutamos `npm install`, cuando termine podremos iniciar el proyecto en desarrollo usando `npm run dev`, y en producción usando `npm run build` y seguidamente `npm run start`.
